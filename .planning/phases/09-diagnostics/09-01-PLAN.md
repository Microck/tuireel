  <action>
Create `packages/core/src/logger.ts` with:

- `LogLevel` enum: `silent`, `normal`, `verbose`, `debug` (ascending verbosity)
- `Logger` class with methods: `info(msg)`, `verbose(msg)`, `debug(msg)`, `step(name, detail?)`, `timing(label, ms)`, `stat(label, value)`
- `info` logs at normal+, `verbose` logs at verbose+, `debug` logs at debug+
- `step` logs formatted step progress: `[step N/M] type: detail` at verbose+
- `timing` logs `label: Xms` at debug level
- `stat` logs `label: value` at verbose level
- All output goes to stderr (not stdout) so it doesn't interfere with piped output
- Export `createLogger(level: LogLevel): Logger`
- A silent no-op logger is the default when no level specified

Export from `packages/core/src/index.ts`: `Logger`, `LogLevel`, `createLogger`
  </action>

**CLI changes (all three commands):**
- Add `--verbose` flag (boolean) and `--debug` flag (boolean) to record, composite, preview commands
- Determine LogLevel: debug flag wins > verbose flag > normal default
- Create logger with `createLogger(level)` and pass to core functions

**Recorder (`record` function signature change):**
- Add optional `logger?: Logger` parameter to `record()` options
- Log at verbose: step execution start/end with step type and index (`[step 2/8] type: launching "node app.js"`)
- Log at verbose: frame count after capture completes (`Captured 342 frames`)
- Log at verbose: encoding start/end (`Encoding MP4...`, `Encoding complete: output.mp4`)
- Log at debug: per-step timing (`step 2 took 1234ms`)
- Log at debug: ffmpeg command being run (full command string)

**Compositor (`compose` function signature change):**
- Add optional `logger?: Logger` parameter to `compose()` options
- Log at verbose: frame decode progress every 100 frames (`Decoding frames: 200/500`)
- Log at verbose: overlay compositing progress every 100 frames
- Log at verbose: final encode stats (frame count, output size)
- Log at debug: ffmpeg encode command and stderr output
- Log at debug: per-frame compositing timing for first 5 frames (to show typical cost)

**Important:** Do NOT change the public API signatures in a breaking way. Logger is always optional with a silent default fallback.
  </action>
