  <name>Task 1: $include resolver with cycle detection</name>
  <files>packages/core/src/config/resolver.ts</files>
  <action>
  Create `resolveIncludes(steps, baseDir, seen?)`:
  - Iterate steps array. For each item:
    - If it has `$include` key: resolve path relative to baseDir, check `seen` Set for cycles (error if found), read+parse the JSONC file, recursively resolve its steps, splice results in place
    - Otherwise: keep step as-is
  - `seen` param tracks the include chain (Set of absolute paths) for cycle detection
  - Return flat Step[] array with all includes resolved

  Handle errors:
  - File not found: "Include file not found: {path} (referenced from {parentFile})"
  - Circular: "Circular include detected: {path} (chain: {seen joined})"
  - Invalid JSON: "Failed to parse include file: {path}: {parseError}"
  </action>
  <verify>Unit test or manual: resolve a config with $include, verify steps are flattened. Circular include produces error.</verify>
  <done>$include resolves recursively with cycle detection. Paths resolve relative to including file.</done>
</task>
