  Create `packages/core/src/utils/debounce.ts`:
  - Export `debounce<T extends (...args: any[]) => any>(fn: T, ms: number): T`
  - Standard debounce: clearTimeout + setTimeout pattern

  Create `packages/core/src/watch.ts`:
  - Export `async function watchAndRecord(configPath: string, options?): Promise<void>`
  - Install chokidar: `pnpm add chokidar --filter @tuireel/core`
  - Watch configPath with chokidar: `chokidar.watch(configPath, { ignoreInitial: true, awaitWriteFinish: { stabilityThreshold: 200 } })`
  - On "change" event (debounced 500ms):
    - If `recording` flag is true, log "Skipping — recording in progress" and return
    - Set `recording = true`
    - Log "Config changed, re-recording..."
    - Load config, call existing `record(config)` function
    - On success: log "✓ Re-recorded successfully"
    - On error: log error but DON'T exit (keep watching)
    - Set `recording = false` in finally block
  - Do initial record on start, then enter watch loop
  - Handle SIGINT to close watcher gracefully
  </action>
  <verify>tsc --noEmit passes. chokidar is in core package.json dependencies.</verify>
  <done>watchAndRecord starts watcher, debounces changes, re-records, handles concurrent protection.</done>
