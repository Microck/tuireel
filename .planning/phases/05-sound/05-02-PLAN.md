  2. Add to `packages/core/src/sound.ts`:
     - `SoundConfig` interface: `{ effects?: SfxConfig; track?: string; trackVolume?: number; effectsVolume?: number }`
     - `buildFullAudioArgs(videoInput, events, durationSec, soundConfig)` — orchestrates:
       - If only effects: use existing `buildAudioMixArgs`
       - If only custom track: single input with duration trim (`-t`), fade out at end (`afade=t=out:st=X:d=2`), volume adjustment
       - If both: build effects track args, add custom track as additional input, `amix=inputs=2` to combine effects+track outputs
       - For custom track shorter than video: pad with silence. For longer: trim with `-t` and fade out.
     - Update finalize functions to accept `SoundConfig` instead of just `SfxConfig`

  3. Add tests for custom track scenarios:
     - Track-only: correct input args, trim to duration, fade out
     - Effects + track combined: correct amix input count, both volumes applied
     - Neither: no audio args generated

  Export `SoundConfig` and `buildFullAudioArgs` from index.
  </action>
