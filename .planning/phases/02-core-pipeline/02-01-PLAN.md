  <files>

    packages/@tuireel/core/src/executor/steps/pause.ts
    packages/@tuireel/core/src/executor/timing.ts
  </files>
  <action>
    Create `TuireelSession` class wrapping tuistory's Session:
    - `createSession(config)`: calls `launchTerminal()` from tuistory with `{command, cols, rows, env}`. Returns TuireelSession.
    - `TuireelSession.screenshot(format?)`: calls `this.session.getTerminalData()` then `renderTerminalToImage(data, {format: 'jpeg', quality: 90})` from `ghostty-opentui/image`. Returns `Buffer`.
    - `TuireelSession.waitIdle()`: delegates to `session.waitIdle()`.
    - `TuireelSession.close()`: delegates to `session.close()`.
    - Expose `session.type()`, `session.press()`, `session.waitForText()`, `session.resize()` via delegation.

    IMPORTANT API details (from actual tuistory source):
    - tuistory's `Session` has NO `.screenshot()` — use `getTerminalData()` + `renderTerminalToImage()`.
    - `launchTerminal()` already calls `waitForData()` + `waitIdle()` before returning.
    - `session.press()` accepts `Key | Key[]` where Key includes modifiers. For chord "ctrl+c", pass `['ctrl', 'c']`.
    - `session.type()` types char-by-char with 1ms delay built in, then waits idle. Our wrapper should NOT use tuistory's built-in type() for human-like typing — instead write chars individually via `session.writeRaw(char)` + custom delay.

    Create individual step handlers in `executor/steps/`:
    - `launch.ts`: Already handled by session creation. Export a no-op or thin validation.
    - `type.ts`: Type text char-by-char using `session.writeRaw(char)` + `charDelay(baseSpeed)` per char, then `session.waitIdle()` at end.
    - `press.ts`: Parse key string (e.g. "ctrl+c" → `['ctrl', 'c']`), call `session.press(keys)`.
    - `wait.ts`: Call `session.waitForText(pattern, {timeout})`. Pattern can be string or regex.
    - `pause.ts`: `await sleep(durationMs)`.

    Create `timing.ts`:
    - `charDelay(baseMs)`: Returns `baseMs * (0.5 + Math.random())` — jittered delay for human-like typing.
    - `sleep(ms)`: Simple promisified setTimeout.
  </action>
  <verify>
    - `tsc --noEmit` passes with no type errors
    - Imports resolve correctly (tuistory, ghostty-opentui/image)
  </verify>
  <done>
    TuireelSession wraps tuistory with screenshot capability. All 5 step types (launch, type, press, wait, pause) have implementations. Typing has human-like jitter.
  </done>
