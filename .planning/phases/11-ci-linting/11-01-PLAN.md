<task type="auto">
  <name>Task 2: GitHub Actions CI workflow</name>
  <files>.github/workflows/ci.yml</files>
  <action>
Create `.github/workflows/ci.yml`:

```yaml
name: CI
on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  ci:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: pnpm/action-setup@v4
      - uses: actions/setup-node@v4
        with:
          node-version: 22
          cache: pnpm
      - run: pnpm install --frozen-lockfile
      - run: pnpm lint
      - run: pnpm exec tsc --noEmit
      - run: pnpm build
      - run: pnpm test
```

Key details:
- Use `pnpm/action-setup@v4` (reads packageManager from package.json)
- Node 22 (LTS, matches project)
- `--frozen-lockfile` for reproducible installs
- Steps in order: install → lint → type-check → build → test
- Single job is fine for this project size (no need for matrix or parallel jobs yet)
- Do NOT install ffmpeg or Sharp platform deps here — that's for the video smoke test job (Phase 11 Plan 02)
  </action>
