    - `bezierControl(x0,y0,x1,y1,dist)` — perpendicular offset control point; skip for short moves (<80px)

    Export only `computeCursorPath` and `moveDuration`. Rest are internal.

    - Path length >= 6 steps
    - Longer distance → more steps
    - humanEase(0)===0, humanEase(1)===1, humanEase(0.4)~=0.5
    - moveDuration increases with distance (monotonic for practical range)
