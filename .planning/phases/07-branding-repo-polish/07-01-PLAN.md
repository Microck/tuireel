Create the `assets/branding/` directory structure.

Design a geometric logo SVG for Tuireel. The icon concept: a stylized film reel frame (rounded rectangle or circle) with a terminal prompt cursor (`>_`) integrated inside, suggesting "terminal recordings." Keep it to 2-3 bold geometric shapes for favicon legibility.

Color palette to establish (from research recommendations):
- Primary: Teal/cyan `#06B6D4` (terminal cursor feel)
- Accent: Warm amber `#F59E0B` (record button association)
- Dark bg: Deep navy `#0F172A`
- Light bg: Off-white `#F8FAFC`

Create three variants:
1. `logo.svg` — Full logo with icon + "tuireel" wordmark text. Viewbox should be wide (e.g., 400x120). Use the primary teal for the icon and dark slate for the text.
2. `logo-dark.svg` — Same layout but colors adjusted for dark backgrounds: light/white text, teal icon with slight brightness boost. Text color `#E2E8F0`.
3. `logo-light.svg` — Same layout but colors adjusted for light backgrounds: dark text `#1E293B`, standard teal icon.

SVG requirements:
- Hand-written SVG (no external fonts — use `<text>` with system fonts or draw text as paths)
- Viewbox-based sizing (no fixed width/height attributes, or use width/height that match viewbox aspect ratio)
- No raster images embedded
- Icon portion must be recognizable when rendered at 16x16, 32x32, and 128x128 pixels
- Keep total SVG under 50 lines each — geometric simplicity is key
