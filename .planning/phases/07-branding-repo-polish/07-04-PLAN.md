<task type="auto">
  <name>Task 1: Create branding generation script</name>
  <files>scripts/generate-branding.ts</files>
  <action>
Create `scripts/generate-branding.ts` — a TypeScript script that uses Sharp (already a dependency of @tuireel/core) to generate both PNG images.

The script should:
1. Import `sharp` from the workspace (it's in packages/core/node_modules, accessible via pnpm workspace)
2. Create a dark background rectangle (`#0F172A` navy) at the target dimensions
3. Read `assets/branding/logo.svg` (the icon-only favicon version for simplicity, or the dark variant)
4. Composite the logo SVG onto the background, centered or offset-left
5. Overlay text for the project name "tuireel" and tagline "Scripted TUI demo recorder" using SVG text overlays (Sharp can composite SVG buffers containing text)
6. Output two files:
   - `assets/branding/banner.png` at exactly 1280x640
   - `assets/branding/og-image.png` at exactly 1200x630

Implementation approach:
- Use Sharp's `create()` to make the background
- Use Sharp's `composite()` to layer the logo SVG and text SVG overlays
- For text rendering, create an inline SVG string with `<text>` elements styled with the brand colors (teal `#06B6D4` for "tuireel", light gray `#E2E8F0` for tagline)
- Use a system-safe font family: `sans-serif` (avoids font installation issues)

Make the script runnable with: `pnpm tsx scripts/generate-branding.ts` or `npx tsx scripts/generate-branding.ts`

Include a shebang or just make it a regular module. Import path via `import { resolve } from 'node:path'` and `import { readFileSync, writeFileSync } from 'node:fs'`.
