    - dumpFrames({ rawVideoPath, outputDirectory, ffmpegPath }) writes PNGs to outputDirectory

    1. Create `packages/core/src/diagnostics/probe-video.ts`:
       - Define `ProbeStream` interface (codec_name, width, height, nb_frames, r_frame_rate, codec_type, duration, etc.)
       - Define `ProbeFormat` interface (filename, duration, size, format_name, etc.)
       - Define `ProbeResult` interface ({ streams: ProbeStream[], format: ProbeFormat })
       - Implement `probeVideo(ffprobePath: string, videoPath: string): Promise<ProbeResult>`
       - Derive ffprobe path from ffmpeg path: `ffmpegPath.replace(/ffmpeg$/, "ffprobe")`
       - Run `ffprobe -v quiet -print_format json -show_format -show_streams <videoPath>`
       - Parse stdout as JSON, return typed result

    2. Create `packages/core/src/diagnostics/inspect.ts`:
       - Define `InspectReport` interface with timeline stats + video probe stats
       - Implement `inspectRecording(options: { timelinePath: string, rawVideoPath: string, ffmpegPath: string }): Promise<InspectReport>`
       - Load timeline via `InteractionTimeline.fromFile(timelinePath)` -> call `.toJSON()` to get TimelineData
       - Compute: frameCount, fps, width, height, keyframeCount (frames.length), eventCount (events.length), terminalFrameCount (terminalFrames?.length ?? 0), durationMs ((frameCount / fps) * 1000)
       - Call `probeVideo()` on the raw video to get codec, raw frame count, raw duration
       - Combine into InspectReport and return
       - Check file existence before loading; throw descriptive errors

    3. Create `packages/core/src/diagnostics/frame-dumper.ts`:
       - Define `DumpFramesResult` interface ({ outputDirectory: string, frameCount: number })
       - Implement `dumpFrames(options: { rawVideoPath: string, outputDirectory: string, ffmpegPath: string, logger?: Logger }): Promise<DumpFramesResult>`
       - Re-implement the decodeRawFrames logic locally (it's not exported from compositor.ts and shouldn't be — keep compositor's internals private). Use the same ffmpeg invocation: `["-y", "-i", rawVideoPath, "-vsync", "0", join(outputDir, "%06d.png")]`
       - Use `ensureFfmpeg()` pattern — accept ffmpegPath directly (caller provides it)
       - After decode, readdir the output directory, count .png files, return result
       - Create outputDirectory with `mkdir -p` if missing
       - Check rawVideoPath exists before running ffmpeg; throw descriptive error
