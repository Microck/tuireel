<task type="checkpoint:human-action" gate="blocking">
  <name>Task 1: Register custom domain in Mintlify dashboard and configure DNS</name>
  <action>
    Two manual steps are required before any code changes:

    **Step A — Mintlify Dashboard:**
    1. Go to https://dashboard.mintlify.com
    2. Navigate to Settings → Deployment → Custom Domain
    3. Enter `tuireel.micr.dev` and save
    4. The dashboard will display DNS instructions (confirming the CNAME target)

    **Step B — DNS Provider (for micr.dev):**
    1. Open your DNS provider (Cloudflare, etc.)
    2. Create a CNAME record:
       - Name: `tuireel`
       - Target: `cname.mintlify-dns.com`
    3. If using Cloudflare specifically:
       - Set SSL/TLS encryption mode to "Full (strict)"
       - Disable "Always Use HTTPS" under SSL/TLS → Edge Certificates
    4. Check for existing CAA records: `dig micr.dev CAA +short`
       - If any CAA records exist, add: `0 issue "letsencrypt.org"`

    **Step C — Wait for DNS propagation:**
    - Check propagation at https://dnschecker.org (search for `tuireel.micr.dev CNAME`)
    - Or run: `dig tuireel.micr.dev CNAME +short` (expect `cname.mintlify-dns.com.`)
    - TLS certificate auto-provisions after DNS propagates (typically 1-24h)

  </action>
  <resume-signal>Type "dns ready" when `dig tuireel.micr.dev CNAME +short` returns `cname.mintlify-dns.com.` and `curl -I https://tuireel.micr.dev` returns a response (200 or redirect, not connection error)</resume-signal>
</task>
