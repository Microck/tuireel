<task type="auto">
  <name>Task 1: Install and configure Changesets</name>
  <files>.changeset/config.json, package.json</files>
  <action>
1. Install changesets:
   ```
   pnpm add -D -w @changesets/cli
   ```

2. Initialize changesets:
   ```
   pnpm exec changeset init
   ```

3. Edit `.changeset/config.json` to configure:
   ```json
   {
     "$schema": "https://unpkg.com/@changesets/config@3.1.1/schema.json",
     "changelog": "@changesets/cli/changelog",
     "commit": false,
     "fixed": [["@tuireel/core", "tuireel"]],
     "linked": [],
     "access": "public",
     "baseBranch": "main",
     "updateInternalDependencies": "patch",
     "ignore": []
   }
   ```

   - `fixed`: Both packages version together (when one bumps, both bump)
   - `access`: "public" for npm publishing
   - `baseBranch`: "main"

4. Add changeset scripts to root `package.json`:
   ```json
   "changeset": "changeset",
   "version": "changeset version",
   "release": "pnpm build && changeset publish"
   ```
  </action>
  <verify>
`ls .changeset/config.json` exists. `cat .changeset/config.json` shows fixed versioning. `pnpm changeset status` runs without error.
  </verify>
  <done>Changesets initialized with fixed versioning group for @tuireel/core and tuireel CLI.</done>
</task>
